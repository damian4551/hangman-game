The game doesn't work on phones Select a letter of the alphabet by typing on keyboard or by using screen keyboard. To hide screen keyboard click „hide keyboard” button in the right corner of the screen. If you are playing on devices with small screens you have to use screen keyboard.

If the letter is contained in word, selected letter displays in right place. If the letter is not contained in word, the part of the hangman's body is added.

The game ends when the whole word is guessed (win) or all the parts of hangman body are displayed (lose).
